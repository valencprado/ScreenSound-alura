# Screen Sound: registro, consulta e avaliação de bandas 🥁

Feito com C# no curso "C#: criando sua primeira aplicação", da Alura.
Conceitos abordados:
- int, strings e List
- Dicionários
- Funções

  Funcionalidades:
  - registrar banda
  - listar bandas
  - avaliar banda já registrada
  - exibir média das avaliações (desafio)
